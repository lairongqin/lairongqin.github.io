32位win7安装运行mongoDB

安装

在win32位的电脑上安装mongoDB server，首先在这个网址上下载相应的安装包https://www.mongodb.com/download-center#previous 新版server已经没有X32位的版本，我们使用旧版本。

安装后，我们在安装文件目录*\MongoDB\Server\3.2\bin里打开cmd命令行，运行mongod命令试一下。

出现问题：

问题一

这个问题是因为mongoDB 3.0版本以后默认使用WiredTiger的存储引擎，因为咱们这个旧版本，所以需要手动更改成旧引擎MMAPv1(当然新引擎肯定好，谁叫咱们用的X32电脑呢)

问题二

win7上一个漏洞需要更新，更新地址点我。

全部搞定后
重新使用命令mongod

监听27017端口，启动成功。

MongoDB shell

MongoDB 自带JavaScript shell, 可在shell中使用命令行与MongoDB实例交互。

步骤：

  1. 运行mongod确保MongoDB服务器运行
  2. 使用命令mongo
