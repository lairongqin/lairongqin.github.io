关于nodeJS的配置

老文，现在简单很多

2015-11-08
#undefined

昨天弄了一下午，关于nodeJS的配置，说白了就是 NVM(版本管理)，nodejs，NPM（包管理）这三样东西。

  1. 通过git克隆NVM（没安装GIT请直接 sudo apt-get install git ）
    cd ~/git

    git clone https://github.com/cnpm/nvm.git

  2. 配置终端启动时自动执行 source ~/git/nvm/nvm.sh通过vi或者vim修改~/.bashrc, ~/.bash_profile, ~/.profile, 或者 ~/.zshrc 文件添加以下命令：

    source ~/git/nvm/nvm.sh

  3. 安装完成后，查看一下nvm ls
  4. 通过
    nvm ls-remote
    查看当前版本
  5. 通过
    nvm install 0.12.0（版本号在之一）
    安装某一版本的nodejs
  6. 安装完成后输入
    nvm use 0.12.0
  7. 便可以使用这一般的nodejs
  8. 这时候再次输入nvm ls就可以查看了

注：通过这一方法安装nodejs已经自带安装了NPM，最后在推荐一本入门书
