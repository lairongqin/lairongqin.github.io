重头再来__JS基本概念

2016-04-18
#JavaScript

基本概念

语法

  • 区分大小写

  • 标识符

    第一个字母必须是一个字母、下划线或一个美元符号；

  • 关键字和保留字

数据类型

简单数据类型
: Undefined、Null、Boolean、Number和string

复杂数据类型
: Object

typeof 操作符

  • “undefined” ——如果这个值未定义
  • “boolean” ——如果这个值是布尔值
  • “sting” ——如果这个值是字符串
  • “number” ——如果这个值是数值
  • “objcet” ——如果这个值是对象或者null
  • “function” ——如果这个值是函数

Undefined 类型

使用var声明但未初始化值时，为undefined。

  • [x] 对为初始化的变量执行 typeof操作符都会返回undefined，对未声明的变量执行typeof操作符同样也会返回undefined值。

Null类型

null 表示一个空对象指针，这也是为何typeof操作符检测null值时返回“objct”

  • [x] 实际上，undefined值是派生自null的，因此 alert(null === undefined); // true 注意这样比较会转换其操作数

Boolean 类型

  • Boolean类型的字面量true和false是区分大小写的
  • 要将一个值转换为Boolean值，可以调用转型函数Boolean()
数据类型 转换为true的值 转换为false的值
Boolean true false
Sring 任何非空字符串 空字符串””
Number 任何非零数字值 0和NaN
Object 任何对象 null
Undefined 无 Undefined

Number类型

  • Number使用IEEE754格式来表示整数和浮点数值
  • 表达八进制值的第一位必须是 0

  • 十六进制字面值的前两位必须是 0x

  • 在进行算数计算时，八进制和十六进制最终将转化成十进制

浮点数值

  • 保存浮点数值需要的内存是保存整数的两倍
  • 对于那些极大或者极小的数值，可以使用e表示法。用e表示法表示的数值等于e前面的数值乘以10的指数次幂
  • 浮点数的最高精度是17位小数，在进行算数计算时精度远远不如整数。

数值范围

  • ES能保存的最小数值保存在 Number.MIN_VALUE 中，在大多数浏览器中，这个值是 5e-324 ；能够表示的最小数值保存在 Number.MAX_VALUE 中，在大多数浏览器中，这个值是1.7976931348623157e+308
  • 如果超出数值范围，会被转化成 Infinity 和 -Infinity
  • 确定一个值是否是有穷的可以使用 isFinite()函数

NaN

  • 任何数值除以非数值都会返回NaN
  • NaN本身有两个非同寻常的特点
    • 任何设置NaN的操作都会返回NaN
    • NaN与任何值都不相等，包括NaN本身

  • 判断NaN可以用isNaN()函数

  • 0除以0返回NaN,正数除以0返回Infinity，负数除以0返回-Infinity

数值转换

  • Number()
    • boolean 返回0或1
    • 数值返回数值
    • null返回0
    • undefined 返回NaN
    • 字符串
      • 值包含数值返回数值
      • 有效浮点值转换浮点值
      • 十六进制转换成十进制
      • 空字符串转为0
      • 其他字符串转为NaN
    • 如果是对象，则调用valueOf()方法，在按上述方法转换，如果结果是NaN则调用toString()在按照上面方法转换。
  • parseInt()
    • 第二个参数为设置转换的进制
  • parseFloat()
    • 只解析十进制，第二个参数无效

string类型

字符字面量

字面量 含义
\n 换行
\t 制表
\b 退格
\r 回车
\f 进纸
\ 斜杠
\xnn 以十六进制代码nnn表示的一个字符
\unnn 以十六进制代码nnnn表示一个Unicode字符
  • 在调用数值的toString()方法可以传递一个参数：输出数值的基数
  • 在不知道要转换的值是不是null或Undefined的情况下，可以调用转型函数String()
    • 如果有toString方法就调用
    • 如果是null则返回“null”
    • 如果是Undefined则返回Undefined

Object类型

每个Object实例都有以下属性和方法

  • constructor()
  • hasOwnProperty()
  • isPrototypeOf()
  • propertyIsEnumerable()
  • toLocaleString()
  • toString()
  • valueOf()

操作符

  • 逻辑非 ！ 无论这个值是什么，都会返回一个布尔值
  • 逻辑与 &&
  • 逻辑或 ||
    • 如果第一个操作数是对象，则返回第一个操作数；
    • 如果第一个操作数的求值结果为false，则返回第二个操作数；
    • 如果两个操作数都是对象，则返回第一个操作数；
    • 如果两个操作数都为null返回null
    • 如果两个操作数都是NaN则返回NaN
    • 如果两个操作数都是Udefined则返回undefined

除法

  • Infinity被Infinity除，结果是NaN；
  • 0除0，结果是NaN；

相等操作符

  • 相等和不相等
    • 如果有一个操作数是布尔值，则在比较相等性之前先将其转换为数值——false转换为0而true转换为1
    • 如果一个操作数是字符串，另一个操作数是数值，在比较相等性之前，先讲字符串转化为数值；
    • 如果一个操作书是对象，另一个操作数不是，则调用对象的valueOf()方法，用得到的基本类型值按照前面的规则进行比较。
    • 在比较相等性之前，不能将null和undefined转换成其他任何值。
    • 如果有一个操作数是NaN，则相等操作符返回false，而不想等操作符返回true。
    • 如果两个操作数都是对象，则比较他们是不是同一个对象。如果两个操作数都指向同一个对象，则相等操作符返回true否则false

全等和不全等

  • undefined == null 返回true undefined === null 返回false

逗号操作符

  • 逗号操作符多用于声明多个变量；但除此之外，逗号操作符还可以用于赋值，逗号操作符总会返回表达式中的最后一项。
